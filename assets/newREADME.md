# ReadMe Generator

## Description
Users can generate a pre-formatted README file using in-line commands!

## Installation 
Users must install inquirer npm before utilizing application.
## Usage 
Spend less time creating a README for your project and simply use in-line commands! This way, you get a quality README file for your repository and can spend more time working on your project.
